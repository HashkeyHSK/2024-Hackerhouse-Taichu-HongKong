# Lazarus - LaBRADOR Prover

## Overview

Lazarus 是一个基于Lattice的ZK-SNARK的Proof System的Rust代码实现框架。我们已经完成了大部分LaBRADOR的Prover逻辑和一部分Verifier的逻辑。演示了目前的代码如何生成证明并进行验证。



## Tech Stack
- Backend: Rust
